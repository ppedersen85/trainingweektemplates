Øktdetaljer

🏃 Hardøkt løp

  • Oppvarming: 15-20 min rolig (puls <155)
  • Hoveddel (velg én):
  • → 4-6 x 4-5 min på terskel (puls ~182)
  • → 20-30 min tempoløp (puls 175-180)
  • → 3 x 8 min progressiv (170-175-180)
  • Pause: 2-3 min aktiv hvile mellom drag
  • Nedjogg: 10-15 min rolig

🚴 Hardøkt sykkel

  • Oppvarming: 20-25 min rolig
  • Hoveddel (velg én):
  • → 4-5 x 8 min på terskel (puls ~172-177)
  • → 2 x 20 min sweetspot (puls 165-170)
  • → 40-60 min tempo (puls 160-165)
  • Pause: 3-4 min lett tråkking
  • Nedkjøring: 15 min rolig

🏃‍♂️ Langtur

  • Varighet: 60-120 min (bygg gradvis)
  • Intensitet: Snakkepuls (puls <155)
  • Terreng: Variert, gjerne kupert
  • Kan være løp eller sykkel
  • Drikke hver 20. min ved >90 min
  • Fokus: Aerob base og fettforbrenning

🚶 Lett økt

  • Varighet: 30-60 min
  • Intensitet: Helt rolig (puls <145)
  • Formål: Aktiv restitusjon
  • Løp, sykkel, svømming eller gange
  • Avslutt med 5-10 min lett tøying
  • Kan kombineres med styrke/stabilitet

💪 Styrketrening (tillegg)

  • Varighet: 20-30 min
  • Fokus: Løpsspesifikk styrke
  • Ettbensknebøy: 3x12 hver side
  • Bulgarsk utfall: 3x10 hver side
  • Planke: 3x45-60 sek
  • Hoftehev: 3x15
  • Tåhev: 3x20

📈 Konkurranseforberedelse

  • Siste 3 uker før konkurranse:
  • Uke -3: Normal belastning
  • Uke -2: 70% volum, økt intensitet
  • Uke -1: 50% volum, kort intensitet
  • Prioriter løpespesifikk trening
  • Reduser sykling gradvis

Treningsprinsipper

Intensitetsfordeling basert på treningsvolum

Timer per uke Rolig trening Hard trening Kommentar
2-3 timer 50-60% 40-50% Høy andel kvalitet for å opprettholde form
4-5 timer 65-70% 30-35% Balansert fordeling, god utvikling
6-8 timer 75-80% 20-25% Klassisk polarisert modell
9+ timer 80-90% 10-20% Elitemodell med høyt volum

Norsk treningsmodell (tilpasset volum)

  • Ved lavt volum (<5 timer/uke): Høyere andel kvalitet (30-40%)
  • Ved moderat volum (5-8 timer/uke): Tradisjonell 80/20 fordeling
  • Ved høyt volum (>8 timer/uke): Kan gå opp til 90/10 fordeling
  • Terskeltrening prioriteres fremfor VO2max for 10km/halvmaraton
  • Doble terskeldager kun ved god restitusjonsevne
  • Progresjon: Øk volum → øk frekvens → øk intensitet

Periodisering (4-ukers sykluser)

  • Uke 1: Moderat belastning (90% av maks)
  • Uke 2: Økt belastning (100%)
  • Uke 3: Toppuke (110% - ekstra hardøkt eller volum)
  • Uke 4: Restitusjon (70% - kun 1 hardøkt)
  • Test/tidløp hver 2. syklus (uke 8, 16, osv)
  • Bygg base i 8-12 uker før spesifikk fase

Restitusjon og tilpasning

  • Minimum 48 timer mellom hardøkter ved <5 økter/uke
  • Kan ha hardøkt 2 dager på rad ved 6+ økter/uke
  • Langtur bør ha minst 1 hviledag før neste hardøkt
  • HRV-måling anbefales for objektiv restitusjonsvurdering
  • Reduser intensitet ved: dårlig søvn, høy hvilepuls (+5 slag), sykdomsfølelse

Løping vs sykling

  • Sykkel: 5-10 slag lavere puls enn løp
  • Tid-konvertering: 1 time løp ≈ 1.5-2 timer sykkel
  • Prioriter løping ved <4 økter/uke
  • Sykkel for ekstra volum uten belastning
  • Maksimalt 40% sykkel i konkurranseperiode

Praktiske tips

  • Loggfør alle økter med puls og opplevd anstrengelse
  • Ukentlig evaluering: justér ved behov
  • Prioriter konsistens over intensitet
  • Bygg volum med maks 10% økning per uke
  • Ta lett uke hver 4. uke uansett hvordan du føler deg
  • Test framgang med 3km/5km test hver 6. uke
